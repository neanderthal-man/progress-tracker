Set up UPSC Prep Board

Your data will be stored on JSONBin — a free cloud service. One-time setup, takes about 3 minutes. After this you get a permanent URL that works on any device, anywhere.

1
Create a free JSONBin account
Go to jsonbin.io → click Sign Up → verify your email.
2
Get your API Key
After logging in, click your profile icon (top right) → API Keys → copy the key shown under "Secret Key". It starts with $2b$.
3
Paste your API key below and connect
We'll automatically create your data storage bin.
